# Menu Restaurante (Expo)

## Rodar o projeto

1. Instale as dependências:

```powershell
# escolha um dos gerenciadores
npm i
# ou
yarn
# ou
pnpm i
```

1. Inicie o app:

```powershell
npm run start
```

1. Escolha a plataforma (Android/iOS/Web) no Expo.

Se usar Android físico, instale o Expo Go e leia o QR code.
